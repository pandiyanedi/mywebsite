Docker Task

Terraform code to do the following tasks:

- use terraform to create the AWS resource (EC2)

- Allow http and ssh ingress using port 80 and 22

- Confirm the EC2 instance is up and running

- Connect the instance by ssh

- Run the shell script to create a repository to store the docker image in docker hub

- Launch an ec2 instance. open port 80 and port 22

- Install and configure docker on the ec2 instance

- And create Dockerfile to get the html from the Github repo to the html folder of the instance

- Get the public dns of the ec2_instance as the out put to test the website
